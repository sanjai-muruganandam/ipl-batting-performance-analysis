# ipl-batting-performance-analysis
Batting Performance Analysis in IPL (2008-2023) using Python, Pandas, and Data Visualization

## Project Overview
This project analyzes batting performance in the Indian Premier League (IPL) using historical data.
Key performance metrics such as runs, batting average, strike rate, and boundary count are explored.

## Objectives
- Analyze IPL batting performance using data analytics
- Compare players based on runs, average, and strike rate
- Identify consistent and aggressive batters
- Visualize insights using charts and graphs

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Exploratory Data Analysis (EDA)
- Top 10 run scorers
- Batting average vs strike rate
- Boundary hitting analysis
- Team-wise total runs
